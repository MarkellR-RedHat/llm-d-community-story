Chapter 1 -- A Project Is Born

The Partner
Network

How 15 organizations came together around a shared vision for open-source LLM inference.

No single company should own the routing plane for LLM serving. Red Hat, Google Cloud, and IBM Research co-founded llm-d with this conviction. NVIDIA and CoreWeave joined as core partners, bringing GPU expertise and scale testing. Together, they built a vendor-neutral inference platform that benefits the entire Kubernetes ecosystem.

  • 3 founding partners set governance and routing/EPP framework
  • 2 core partners contributed GPU optimization and scale testing
  • 6 ecosystem partners bring hardware and model support
  • 3 academic/governance institutions anchor research and IP

Chapter 3 -- The Ecosystem

Cloud-Native
AI Landscape

llm-d connects to the broader cloud-native AI ecosystem through 8+ integrations.

  • vLLM -- High-throughput inference engine with PagedAttention (v0.6+)
  • KServe -- Model serving platform on Kubernetes (v0.13+)
  • Gateway API -- Kubernetes-native routing with Inference Extension (v1.2+)
  • NIXL -- NVIDIA GPU-to-GPU KV cache transfer library (v0.3+)
  • Kubernetes -- Container orchestration with CRDs and operators (v1.28+)
  • Prometheus -- Observability, autoscaling metrics (v2.45+)
  • KEDA -- Event-driven autoscaling by queue depth (v2.12+)
  • Envoy -- Data plane for Gateway API traffic (v1.28+)
